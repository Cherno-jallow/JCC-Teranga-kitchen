# Teranga Kitchen - Style Guide

## Layout

The design uses a max-width container:

- Container max-width: 1000px
- Container padding: 0 20px

## Colors

### Primary

- **Primary (Terracotta Orange)**: #C2571A
- **Accent (Gold)**: #D4A84B

### Neutral

- **Dark Brown**: #2C1810
- **Header/Navbar**: #1B0801
- **Text**: #333333
- **Secondary (Warm Cream)**: #F5E6D3
- **Background**: #FDF8F3
- **White**: #FFFFFF

## Typography

### Body Copy

- Font size: 16px
- Line height: 1.6

### Headings

- H1: 32px (36px in hero)
- H2: 24px
- H3: 20px

### Font

- Family: Georgia, serif
- Weights: 400 (normal), 700 (bold)

## Spacing

- Section margin-bottom: 40px
- Paragraph margin-bottom: 15px
- Container padding: 20px
- Header/Footer padding: 15px / 25px

## Components

### Buttons

- Background: #C2571A
- Text: #FFFFFF
- Padding: 12px 25px
- Border radius: 5px
- Hover background: #D4A84B
- Hover text: #2C1810

### Cards/Features

- Background: #F5E6D3
- Padding: 20px
- Border radius: 8px

### Forms

- Input padding: 12px
- Border: 2px solid #F5E6D3
- Border radius: 5px
- Focus border: #C2571A
